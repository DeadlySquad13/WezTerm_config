# WezTerm Dotfiles
WezTerm configuration files.

## Installation
```sh
# For X11/Wayland:
cd $XDG_CONFIG_HOME/wezterm
# For all other systems:
cd $HOME/.config/wezterm

git clone <repoUrl> .
```
